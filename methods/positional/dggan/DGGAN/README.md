# DGGAN

The repository contains refernce implementation of DGGAN proposed in 
Adversarial Directed Graph Embedding. 
Shijie Zhu, Hao Peng, Jianxin Li, Senzhang Wang, Lifang He.
In AAAI 2021.
https://arxiv.org/abs/2008.03667


## Requirements

- tensorflow
- numpy
- scikit-learn


## Usage

```bash
cd code
python dggan.py
```
